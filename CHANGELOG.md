# 更新记录

## 1.0.0-beta.4 - 2026-07-17

### 新增

- DOCX 改为原生 Office Open XML 文档，文字、标题、列表、表格、链接、颜色和代码块可继续编辑。
- PDF 与 DOCX 统一采用 A4 页面尺寸，PDF 按页展示，文档默认显示比例为 100%。
- 分屏模式支持编辑区与预览区双向同步滚动。
- 帮助菜单联系邮箱更新为 `lproc2019@gmail.com`。

### 优化

- 预览严格保留编辑内容中的回车次数，一次回车对应一次换行，连续回车保留对应空行。
- 隐私政策移至“清笺MD”应用菜单，降低工具栏误触概率。
- 系统菜单栏精简为“清笺MD”和“帮助”；文件及格式操作继续由窗口工具栏提供。
- 修复无 Storyboard 启动时应用菜单被 macOS 重置的问题。
- 新建文稿和空白文件默认进入编辑模式，已有内容的文稿默认进入预览模式。
- 分屏同步改为按 Markdown 源码行与正文段落定位，确保两侧当前显示内容持续对应。
- 编辑状态开始搜索时自动隐藏选区快捷格式条，关闭搜索后按当前选区恢复。
- 分屏模式新增当前编辑行联动高亮，预览区以亮黄色标出鼠标悬停或输入光标所在源码行对应内容。
- 首个独立 Markdown 文稿窗口打开时自动居中，已有标签页窗口保持用户调整后的位置。

## 1.0.0-beta.3 - 2026-07-15

### 新增

- 新增显示比例面板，支持常用比例、连续滑杆及加减按钮。
- 预览模式中的任务清单现在可以直接勾选或取消，并同步保存到 Markdown 原文。

### 优化

- 浮动快捷格式条默认贴近选中文字出现，并保留自由拖动能力。
- 快捷颜色精简为红、橙、绿、蓝、紫五种常用颜色。
- 快捷颜色末尾新增“取消颜色”，支持恢复选中文字的默认颜色。
- 顶部保存、另存为和导出按钮整合为“文件”下拉菜单，包含新建、打开、保存、另存为、PDF 与 DOCX 导出。
- 分屏模式的编辑区与预览区共用同一显示比例和基准字号。

## 1.0.0-beta.2 - 2026-07-15

### 优化

- 分屏模式下，编辑区与预览区改为分别独立滚动。
- 保留预览内容即时刷新，并分别保持两侧当前阅读位置。

## 1.0.0-beta.1 - 2026-07-15

首个公开测试版。

### 功能

- Markdown 编辑、预览和分屏模式
- 实时预览与分屏模式
- 常用 Markdown 格式和浮动快捷格式条
- 五种快捷颜色
- 查找、替换、结果计数与定位
- 自动保存、另存为、最近文件和多标签页
- PDF 与 DOCX 导出
- 深色和浅色外观自动适配
- Apple 芯片与 Intel Mac 通用构建

### 已知限制

- 测试版尚未使用 Apple Developer ID 签名和公证，首次打开需要在 macOS“隐私与安全性”中手动允许。
- 暂未提供应用内自动更新，请从 GitHub Releases 手动下载新版本。
- 颜色使用清笺MD扩展语法，其他 Markdown 编辑器可能不会渲染颜色。
