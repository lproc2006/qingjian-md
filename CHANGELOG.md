# 更新记录

## 1.0.0-beta.3 - 2026-07-15

### 新增

- 新增显示比例面板，支持常用比例、连续滑杆及加减按钮。
- 预览模式中的任务清单现在可以直接勾选或取消，并同步保存到 Markdown 原文。

### 优化

- 浮动快捷格式条默认贴近选中文字出现，并保留自由拖动能力。
- 快捷颜色精简为红、橙、绿、蓝、紫五种常用颜色。
- 快捷颜色末尾新增“取消颜色”，支持恢复选中文字的默认颜色。
- 顶部保存、另存为和导出按钮整合为“文件”下拉菜单，包含新建、打开、保存、另存为、PDF 与 DOCX 导出。
- 分屏模式的编辑区与预览区共用同一显示比例和基准字号。

## 1.0.0-beta.2 - 2026-07-15

### 优化

- 分屏模式下，编辑区与预览区改为分别独立滚动。
- 保留预览内容即时刷新，并分别保持两侧当前阅读位置。

## 1.0.0-beta.1 - 2026-07-15

首个公开测试版。

### 功能

- Markdown 编辑、预览和分屏模式
- 实时预览与分屏模式
- 常用 Markdown 格式和浮动快捷格式条
- 五种快捷颜色
- 查找、替换、结果计数与定位
- 自动保存、另存为、最近文件和多标签页
- PDF 与 DOCX 导出
- 深色和浅色外观自动适配
- Apple 芯片与 Intel Mac 通用构建

### 已知限制

- 测试版尚未使用 Apple Developer ID 签名和公证，首次打开需要在 macOS“隐私与安全性”中手动允许。
- 暂未提供应用内自动更新，请从 GitHub Releases 手动下载新版本。
- 颜色使用清笺MD扩展语法，其他 Markdown 编辑器可能不会渲染颜色。
