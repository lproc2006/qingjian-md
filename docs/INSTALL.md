# 安装说明

## 系统要求

- macOS 14.0 或更高版本
- 支持 Apple 芯片和 Intel 处理器

## 安装步骤

1. 从 [Releases](https://github.com/lproc2006/qingjian-md/releases/latest) 下载最新 ZIP 安装包。
2. 双击 ZIP 文件解压。
3. 将“清笺MD”拖入 Finder 的“应用程序”文件夹。
4. 在“应用程序”中双击“清笺MD”。

## 首次打开的安全提示

当前公开测试版尚未取得 Apple Developer ID 签名和公证。macOS 首次打开时可能显示“无法验证开发者”或“Apple 无法检查它是否包含恶意软件”。

请仅从本仓库的 Releases 页面下载安装包，并核对发布页提供的 SHA-256 校验值。确认来源后：

1. 先尝试打开一次清笺MD，然后关闭系统提示。
2. 打开“系统设置 > 隐私与安全性”。
3. 向下滚动到“安全性”，找到被阻止的清笺MD。
4. 点按“仍要打开”，输入 Mac 登录密码并确认。

系统会把清笺MD保存为例外，之后可以正常双击打开。Apple 官方说明：[打开来自未知开发者的 Mac App](https://support.apple.com/zh-cn/guide/mac-help/mh40616/mac)。

## 设为 Markdown 默认应用

1. 在 Finder 中选中任意 `.md` 文件。
2. 按 `Command-I` 打开“显示简介”。
3. 展开“打开方式”，选择“清笺MD”。
4. 点按“全部更改”并确认。

此后双击 `.md` 或 `.markdown` 文件即可直接使用清笺MD打开。

## 更新

测试版暂未提供应用内自动更新。新版本发布后，请从 Releases 下载并用新版本替换“应用程序”中的旧版本。已保存的 Markdown 文稿不会因此被删除。

## 卸载

退出清笺MD后，将“应用程序”中的清笺MD移到废纸篓即可。应用不会把文稿集中存入专用资料库；删除应用前仍应确认自己的文稿已保存在所需位置。
