# 使用指南

## 新建与打开

- 选择“文件 > 新建”创建空白 Markdown 文稿。
- 选择“文件 > 打开”打开 `.md` 或 `.markdown` 文件。
- 也可以从 Finder 双击 Markdown 文件直接打开。
- 新建文稿或空白文件默认进入编辑模式；已有内容的文稿默认进入预览模式。

## 三种视图

窗口右上角提供“预览、编辑、分屏”三个模式：

- **预览**：阅读排版后的最终效果。
- **编辑**：专注输入 Markdown 原文。
- **分屏**：左侧编辑、右侧预览，修改会实时显示。

分屏模式下编辑区与预览区可以分别独立滚动，适合一边修改前文，一边查看其他位置的最终排版。

## Markdown 格式

工具栏的“格式”菜单包含：

- 一级至六级标题
- 加粗、斜体、删除线、行内代码
- 无序列表、有序列表、任务清单
- 引用、代码块、分隔线
- 链接、图片、表格、脚注
- 常用文字颜色与自定义颜色

选择文字后会显示浮动快捷格式条。连续应用同类格式时，清笺MD会更新当前格式，避免重复包裹相同标记。浮动工具条可拖动到不遮挡内容的位置。

## 先选格式再输入

光标位于空白位置时，可先选择加粗、斜体、链接、代码等格式，再输入内容。清笺MD会放置格式标记并选中占位文字，输入内容会替换占位文字。

## 颜色

颜色属于清笺MD扩展语法，例如：

```text
[color=#0A84FF]蓝色文字[/color]
```

清笺MD预览可以显示该颜色。其他只支持标准 Markdown 的编辑器可能把颜色标记显示为普通文本。

## 查找与替换

- 在编辑、预览和分屏模式下均可使用查找。
- 搜索栏显示当前结果位置和结果总数。
- 使用向上、向下按钮在结果之间移动。
- 编辑模式下可以替换当前结果或全部替换。
- 点按搜索栏右侧关闭按钮即可退出查找。

## 保存与自动保存

- 已保存过的文稿支持 macOS 自动保存。
- “文件 > 保存”保存当前文稿。
- “文件 > 另存为”保留原文件并创建新副本。
- “文件 > 最近打开”可快速重新打开近期文稿。

## 多标签页

同时打开多个 Markdown 文件时，窗口会优先使用 macOS 原生标签页。可从“窗口”菜单切换标签页、将标签页移到新窗口，或合并所有窗口。

## 导出

工具栏“导出”菜单支持：

- **PDF**：适合阅读、打印和分享最终版式。
- **DOCX**：适合交给 Word 或 WPS 继续编辑。

导出不会修改原 Markdown 文件。

## 缩放与全屏

工具栏“缩放”菜单可以放大、缩小或恢复实际大小。编辑区与预览区会采用一致的缩放比例。使用 macOS 全屏快捷键可进入或退出全屏。

## 链接

预览中的网页链接会交给 macOS 默认浏览器打开。打开外部链接前，请确认链接来源可信。
