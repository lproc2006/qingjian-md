# 使用指南

## 新建与打开

- 选择“文件 > 新建”创建空白 Markdown 文稿。
- 选择“文件 > 打开”打开 `.md` 或 `.markdown` 文件。
- 也可以从 Finder 双击 Markdown 文件直接打开。
- 新建文稿或空白文件默认进入编辑模式；已有内容的文稿默认进入预览模式。
- 首个独立文稿窗口默认位于当前电脑屏幕中央；后续文稿继续使用原生标签页，不会移动已经手动调整过的窗口。

## 三种视图

窗口右上角提供“预览、编辑、分屏”三个模式：

- **预览**：阅读排版后的最终效果。
- **编辑**：专注输入 Markdown 原文。
- **分屏**：左侧编辑、右侧预览，修改会实时显示。

分屏模式下滚动编辑区或预览区时，另一侧会按当前正文段落同步定位，确保两边当前显示的内容始终对应，便于持续对照原文和最终排版。

在编辑区点击或移动输入光标时，光标所在源码行对应的预览内容会显示亮黄色背景。预览实时刷新、滚动或切换分屏后，高亮仍会跟随当前编辑行。

普通文字会严格保留回车次数：一次回车换一行，两次回车保留一行空白，更多连续回车也会保留相同的换行数量。

## 文件工具栏

点按顶部“文件”可打开常用文件操作：

- 新建与打开 Markdown 文稿
- 保存与另存为
- 导出为 PDF 或 DOCX

顶部不再分散显示多个保存和导出按钮，文件操作统一集中在这一处。

## Markdown 格式

工具栏的“格式”菜单包含：

- 一级至六级标题
- 加粗、斜体、删除线、行内代码
- 无序列表、有序列表、任务清单
- 引用、代码块、分隔线
- 链接、图片、表格、脚注
- 红、橙、绿、蓝、紫五种常用文字颜色

选择文字后，浮动快捷格式条会贴近选区出现。连续应用同类格式时，清笺MD会更新当前格式，避免重复包裹相同标记。五种颜色后的叉号按钮可取消选中文字的颜色。浮动工具条可拖动到不遮挡内容的位置；取消选择后，下次选择文字会重新贴近新的选区。

## 先选格式再输入

光标位于空白位置时，可先选择加粗、斜体、链接、代码等格式，再输入内容。清笺MD会放置格式标记并选中占位文字，输入内容会替换占位文字。

## 颜色

颜色属于清笺MD扩展语法，例如：

```text
[color=#0A84FF]蓝色文字[/color]
```

清笺MD预览可以显示该颜色。其他只支持标准 Markdown 的编辑器可能把颜色标记显示为普通文本。

## 任务清单

使用 `- [ ]` 创建未完成任务，使用 `- [x]` 创建已完成任务。在预览或分屏模式中可以直接点按待办框，清笺MD会即时更新编辑区中的 Markdown 标记并自动保存。

## 查找与替换

- 在编辑、预览和分屏模式下均可使用查找。
- 搜索栏显示当前结果位置和结果总数。
- 使用向上、向下按钮在结果之间移动。
- 编辑模式下可以替换当前结果或全部替换。
- 点按搜索栏右侧关闭按钮即可退出查找。
- 编辑状态开始搜索时，选区快捷格式条会自动隐藏，避免遮挡搜索结果；关闭搜索后，如仍有文字被选中，快捷格式条会重新出现。

## 保存与自动保存

- 已保存过的文稿支持 macOS 自动保存。
- “文件 > 保存”保存当前文稿。
- “文件 > 另存为”保留原文件并创建新副本。
- “文件 > 最近打开”可快速重新打开近期文稿。

## 多标签页

同时打开多个 Markdown 文件时，窗口会优先使用 macOS 原生标签页。可直接点按标签页切换文稿，也可使用系统标签页快捷键切换。

## 导出

工具栏“文件”菜单支持：

- **PDF**：采用 A4 页面分页导出，默认显示比例为 100%，适合阅读、打印和分享最终版式。
- **DOCX**：采用 A4 原生 Word 结构，文字、标题、列表、表格和链接可在 Word、WPS 或 LibreOffice 中继续编辑。

导出不会修改原 Markdown 文件。

## 缩放与全屏

点按工具栏中的百分比可打开“显示比例”面板。可以选择常用比例、拖动连续滑杆，或使用减号和加号逐级调整。当前百分比始终显示在工具栏中；编辑区与预览区共用同一基准字号和显示比例。使用 macOS 全屏快捷键可进入或退出全屏。

## 链接

预览中的网页链接会交给 macOS 默认浏览器打开。打开外部链接前，请确认链接来源可信。
