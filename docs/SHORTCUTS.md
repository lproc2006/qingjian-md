# 键盘快捷键

清笺MD遵循 macOS 常用快捷键。

## 文件

| 操作 | 快捷键 |
| --- | --- |
| 新建 | `Command-N` |
| 打开 | `Command-O` |
| 保存 | `Command-S` |
| 另存为 | `Command-Shift-S` |
| 关闭窗口或标签页 | `Command-W` |
| 导出 PDF | `Command-Shift-P` |
| 导出 DOCX | `Command-Shift-D` |

## 编辑

| 操作 | 快捷键 |
| --- | --- |
| 撤销 | `Command-Z` |
| 重做 | `Command-Shift-Z` |
| 剪切 | `Command-X` |
| 拷贝 | `Command-C` |
| 粘贴 | `Command-V` |
| 粘贴并匹配样式 | `Command-Option-Shift-V` |
| 全选 | `Command-A` |
| 加粗 | `Command-B` |
| 斜体 | `Command-I` |
| 插入链接 | `Command-K` |

## 查找

| 操作 | 快捷键 |
| --- | --- |
| 查找 | `Command-F` |
| 查找并替换 | `Command-Option-F` |
| 查找下一个 | `Command-G` |
| 查找上一个 | `Command-Shift-G` |

## 显示与窗口

| 操作 | 快捷键 |
| --- | --- |
| 放大 | `Command-+` |
| 缩小 | `Command--` |
| 实际大小 | `Command-0` |
| 进入或退出全屏 | `Control-Command-F` |
| 上一个标签页 | `Command-Shift-{` |
| 下一个标签页 | `Command-Shift-}` |
| 最小化 | `Command-M` |
| 退出清笺MD | `Command-Q` |
