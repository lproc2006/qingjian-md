# 清笺MD 1.0.0 Beta 4

本次测试版重点完善导出质量、分屏对照、换行显示和 macOS 菜单体验。

## 亮点

- Finder 双击 `.md` 或 `.markdown` 文件即可打开
- 首个独立文稿窗口打开后自动居中显示
- 预览、编辑、分屏三种模式一键切换
- 分屏实时预览，滚动任一侧时另一侧按当前正文段落同步定位
- 分屏中鼠标悬停或输入光标所在行会在预览区以亮黄色同步标出
- 编辑中的回车次数会在预览中原样保留，连续空行不再被折叠
- 比例预设、连续滑杆、加减按钮组成的显示比例面板
- 预览模式可直接勾选任务清单，并同步更新 Markdown 原文
- 快捷格式条贴近选区出现，提供五种常用颜色与一键取消颜色
- 编辑状态开始搜索时自动隐藏快捷格式条，避免遮挡搜索结果
- 新的“文件”下拉菜单集中管理新建、打开、保存、另存为和导出
- macOS 原生窗口、多标签页、自动保存和键盘快捷键
- 完整常用 Markdown 格式与可移动快捷格式条
- PDF 按 A4 页面分页导出，默认显示比例为 100%
- DOCX 使用 A4 原生 Word 结构，文字、表格和链接可继续编辑
- 精简系统菜单栏，仅保留“清笺MD”和“帮助”
- 帮助菜单联系邮箱更新为 `lproc2019@gmail.com`
- 深色与浅色模式自动适配
- Apple 芯片与 Intel Mac 通用安装包

## 下载

下载 `QingjianMD-1.0.0-beta.4-macOS-universal.zip`，解压后将“清笺MD”拖入“应用程序”文件夹。

## 首次打开

当前测试版尚未使用 Apple Developer ID 签名和公证。首次打开时，请先尝试打开一次应用，再前往“系统设置 > 隐私与安全性”，点按“仍要打开”。请只从本发布页下载安装包，并核对 `SHA256SUMS.txt` 中的校验值。

完整说明请阅读仓库首页的[安装说明](https://github.com/lproc2006/qingjian-md/blob/main/docs/INSTALL.md)与[使用指南](https://github.com/lproc2006/qingjian-md/blob/main/docs/USER_GUIDE.md)。
